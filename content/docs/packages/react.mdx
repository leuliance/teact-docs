```bash
bun add @teactjs/react
```

## Highlights

- **Messaging** — `Message`, formatting helpers, `Notification`
- **Keyboards** — `InlineKeyboard`, `Button`, `ButtonRow`, `WebAppButton`, `ReplyKeyboard`, `ReplyButton`, …
- **Media** — `Photo`, `Video`, `Document`, `Sticker`, `MediaGroup`, …
- **Other** — `Poll`, `Contact`, `Location`, `ErrorBoundary`, `SuspenseFallback`
- **Data** — `useQuery`, `useMutation`

Full examples live in [UI components](/docs/guides/components) and the package README in the repo.

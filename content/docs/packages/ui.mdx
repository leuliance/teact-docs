---
title: "@teactjs/ui"
description: Re-exports every component and hook from @teactjs/react under a UI-focused path.
---

```bash
bun add @teactjs/ui
```

```tsx
import { Message, Button, useQuery } from "@teactjs/ui";
```

Functionally **identical** to `@teactjs/react`. Pick whichever import style fits your codebase.
